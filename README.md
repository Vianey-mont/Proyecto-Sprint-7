# Aplicación de Análisis de Datos - Sprint 7

### ¿Para qué sirve?
Esta aplicación web permite visualizar datos mediante gráficas interactivas. Es un proyecto desarrollado para practicar el despliegue de herramientas de ciencia de datos.

### Funcionalidades
* Generación de histogramas a partir de un conjunto de datos.
* Creación de gráficas de dispersión (scatter plots) para comparar variables.
* Interfaz construida con Streamlit/Python.# Proyecto-Sprint-7
